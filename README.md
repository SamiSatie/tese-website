# TeSE website - bilingual static version

This repository contains a bilingual static website for the TeSE research team.

## Structure

- `/fr/`: French version
- `/en/`: English version
- `/assets/css/styles.css`: shared design
- `/assets/js/main.js`: mobile navigation
- `index.html`: automatic redirect to `/fr/`

## GitHub Pages deployment

1. Create a public GitHub repository, for example `tese-site`.
2. Upload all files at the root of the repository.
3. Open `Settings > Pages`.
4. Select `Deploy from a branch`, branch `main`, folder `/root`.
5. The site will be available at `https://your-username.github.io/tese-site/`.
